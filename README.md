# chat

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

项目结构
```
├── public
├── src                   // 源码
│   ├── assets            // 静态资源

│   ├── router            // 路由

│   ├── App.vue
│   └── main.js
├── .gitignore

```
